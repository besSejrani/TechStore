# Installation

<hr/>

## Download Project

=== "Git"

```
git clone git@github.com:besSejrani/BlueberryShop.git

```

<hr/>

## Install Dependencies

=== "Client"

    ```bash
    // NPM
    cd Client
    npm install
    ```

    ```bash
    // Yarn
    cd Client
    yarn install

    ```

=== "Server"

    ```bash
    // NPM
    cd Server
    npm install
    ```

    ```bash
    // Yarn
    cd Server
    yarn install

    ```

<hr/>

## Run Application

=== "Script"

```bash
// NPM
cd Server
npm run dev
```

```bash
// Yarn
cd Server
yarn run dev
```
